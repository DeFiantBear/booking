SVG to PNG Converter for Farcaster Mini App

Convert Your SVG Images to PNG

Farcaster requires PNG images for proper Mini App embedding. Use this tool to convert your SVG files.

Embed (3:2 ratio)


Manual Conversion Instructions:

  1. Use the buttons above to convert each SVG to PNG
  2. Right-click on each converted image and "Save image as..."
  3. Save the PNG files in your public/ folder with these names:
    • icon.png (1024x1024)
    • splash.png (200x200)
    • hero.png (1200x630)
    • miniapp-embed.png (1200x800)
  4. Replace the SVG files with the PNG versions
  5. Deploy your app to make the changes live

Alternative: Use online tools like: Convertio or CloudConvert
